Samsara for Business

Samsara is free and open source for individuals and organisations. This page is for teams evaluating it as an accessibility tool or interested in a future supported deployment.

Why businesses use Samsara

Accessibility and accommodation duties vary by jurisdiction and situation. For some employees with repetitive-strain injuries, chronic pain, or motor disabilities, reducing keyboard and mouse use can be a meaningful part of an accommodation plan.

Samsara is open source, keeps its core dictation and commands local, and can be adapted to workplace tools through its plugin system and extensible command set. It is still beta software and should be evaluated against the reliability, privacy, security, and support requirements of the intended environment.

Possible pilot areas

Installation and configuration

Evaluate Samsara on a small number of workstations and document the required microphone, audio, display, and application configuration.

Custom plugin development

Assess whether a workplace tool can be reached safely through Samsara's existing command and plugin interfaces.

Custom command packs

Prototype a bounded set of commands for a specific workflow and test them with the people who will actually use them.

Training and onboarding

Document hands-free, dictation, microphone, and command workflows for a pilot group.

About

Samsara is built by a solo developer with 10 years in IT systems administration. I built it because I needed it. Chronic pain in my hands made it increasingly difficult to work at a keyboard all day. The existing tools were either too expensive, too limited, or too dependent on cloud services.

I use Samsara every day. That lived experience shapes the product, but organisational deployment and support are still being developed rather than sold as a mature service.

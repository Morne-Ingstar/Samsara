# AI Prompt Templates for Samsara

These templates help you use AI assistants (Claude, ChatGPT, etc.) to customize Samsara without needing to code.

## Available Templates

| Template | Use Case |
|----------|----------|
| [create-voice-command.md](create-voice-command.md) | Add custom voice commands |
| [create-profile.md](create-profile.md) | Create domain-specific vocabulary profiles |
| [create-macro-chain.md](create-macro-chain.md) | Build multi-step command sequences |
| [troubleshoot.md](troubleshoot.md) | Diagnose and fix issues |
| [accessibility-tuning.md](accessibility-tuning.md) | Customize for specific accessibility needs |

## How to Use

1. **Open the template** you need
2. **Copy the prompt section** (everything below the "---" line)
3. **Paste into your AI** (Claude, ChatGPT, Gemini, etc.)
4. **Fill in your request** where indicated
5. **Apply the output** to Samsara as instructed

## Tips for Best Results

### Be Specific
❌ "Add some gaming commands"  
✅ "Add commands for Minecraft: place block, break block, open inventory, sprint"

### Provide Context
❌ "Fix my transcription"  
✅ "Whisper keeps hearing 'react' as 'react to' - I'm a web developer"

### Iterate
The first output might not be perfect. Ask the AI to:
- "Make the delays longer"
- "Add more commands for X"
- "Simplify the trigger phrases"
